## Image Widget
Format yang support
- JPEG
- PNG
- GIF
- WEBP
- BMP
- WBMP

Jenis provider image
- Asset : dari aplikasi
- Network : dari internet
- File : dari memory internal / external
- Memory : raw data

Image harus terdaftar pada pubspec.yaml
